# hibi-zakka — design-vintage-sundries-zakka

> **"Tiny flutters in everyday life, through old little things."**

A design study exploring a fictional small zakka (sundries) shop in Nakameguro that mixes vintage finds with handcraft — targeting **a 26-year-old apparel-shop worker who collects kokeshi, old textiles, and writing tools**.

hibi-zakka is a fictional shop created for this design study.

## Overview

| | |
|---|---|
| **Brand** | hibi-zakka (日々雑貨) |
| **Persona** | zakka (Sundries / handcraft) |
| **Live Site (planned)** | `design.vintage-sundries-zakka.riumu.net` |

## Design Concept

- **Color**: Unbleached linen `#f7f1e8` × hemp `#d4c3a5` × matcha `#697764` × vermilion `#c4513a` × peach-blossom `#d9a89d`
- **Typography**: Klee One (display/body) × BIZ UDPMincho (subhead) × Yuji Syuku (accents)
- **Aesthetic**: Natural wa-modern × handcraft × kraft paper collage
- **UX**: Polaroids with masking tape, notepad ruled background, monthly events block, hanko-stamped greetings
- **Texture**: Kraft paper, washi tape stripes, highlighter swipes

## Tech Stack

- Pure HTML + CSS Custom Properties + Vanilla JS
- Google Fonts CDN
- No framework, no build step — GitHub Pages ready

# 日々雑貨 — design-vintage-sundries-zakka（日本語）

> **「日々の小さなときめきを、古い雑貨で。」**

ヴィンテージ雑貨 ・ デザイン研究シリーズの雑貨総合ペルソナ。**26歳・手しごと好きのアパレル店員**——こけし・古布・古い手紙の道具を少しずつ集めて、毎日の暮らしに小さなときめきを加えたい層——に特化した、中目黒の架空店舗です。

日々雑貨は本デザイン研究のために作成した架空店舗です。
